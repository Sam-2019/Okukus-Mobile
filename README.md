## Getting Started

Please follow the instructions below to set up this project locally.

### Installation

1. Clone the repo

```
git clone https://github.com/Sam-2019/Okukus-Mobile.git
```

via GitHub CLI

```
gh repo clone Sam-2019/Okukus-Mobile
```

2. Install NPM packages

```
npm install
```
